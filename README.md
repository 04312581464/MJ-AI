## üöÄ Installation Guide

### üìã Prerequisites

#### **System Requirements:**
- **OS:** Windows 10/11 (primary), macOS, Linux
- **Python:** 3.8 or higher
- **RAM:** 8GB minimum (16GB recommended)
- **Storage:** 5GB free space
- **Internet:** Required for AI features

#### **Software Dependencies:**
- Git
- Python 3.8+
- Flutter (for mobile app)
- Android Studio (for mobile development)

### üêç Python Environment Setup

#### **1. Clone Repository**
```bash
git clone https://github.com/your-username/MJ-AI.git
cd MJ-AI
```

#### **2. Create Virtual Environment**
```bash
python -m venv mj_ai_env
# Windows
mj_ai_env\Scripts\activate
# macOS/Linux
source mj_ai_env/bin/activate
```

#### **3. Install Dependencies**
```bash
pip install -r requirements.txt
```

#### **4. Environment Configuration**
```bash
# Copy environment template
cp .env.example .env

# Edit .env file with your API keys
# Required keys:
# - GOOGLE_API_KEY (for Gemini)
# - OPENAI_API_KEY (for ChatGPT)
# - LIVEKIT_URL, LIVEKIT_API_KEY, LIVEKIT_API_SECRET
```

#### **5. Verify Installation**
```bash
python agent.py console
```

### üîß Common Issues

#### **Installation Problems:**

#### **API Key Issues:** (If needed)
```bash

python -c "from dotenv import load_dotenv; load_dotenv(); import os; print('API Key:', os.getenv('OPENAI_API_KEY')[:10] + '...' if os.getenv('OPENAI_API_KEY') else 'Not found')"
```
